# 🐾 PETCARE – AI Powered Pet Adoption & Management Platform

## 🌐 Overview

**PETCARE** is a full-stack MERN web application designed to simplify pet adoption, pet management, and pet care services. It connects pet lovers, adoption centers, and pet communities in a single platform, enhanced with an AI chatbot for smart assistance.

---

## 🚀 Key Features

### 🏠 Pet Adoption System

* Browse pets available for adoption
* Connect with nearby pet adoption centers
* View detailed pet profiles

---

### 🛒 Pet Buying & Selling

* List pets for sale
* Explore available pets
* Connect buyers and sellers

---

### 🩺 Pet Health & Management

* Vaccination reminders
* Pet health tracking
* Consultation support

---

### 🍖 Pet Food & Essentials

* Pet food listings
* Recommendations based on pet needs

---

### 👥 Pet Community

* Connect with pet owners
* Ask questions and share experiences
* Community-based discussions

---

### 🤖 AI Chatbot (Gemini API)

* Smart assistant for pet-related queries
* Instant guidance for adoption, care, and health
* 24/7 AI support

---

## ⚙️ Installation & Setup

```bash id="h3g1xk"
git clone https://github.com/Sobika-E/PETCARE.git
cd PETCARE
```

### 2️⃣ Install Dependencies

```bash id="r8k2qp"
npm install
```

### 3️⃣ Start Backend

```bash id="w9n5ls"
npm run server
```

### 4️⃣ Start Frontend

```bash id="t2m6zd"
npm start
```
